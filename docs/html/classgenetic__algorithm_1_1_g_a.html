enum genetic_algorithm::GA::Mode
strong

The type of the genetic algorithm used, depending on the problem type (single- or multi-objective optimization).
Set the mode used with mode.

Enumerator
single_objective 

Simple single-objective genetic algorithm.

multi_objective_sorting 

Non-dominated sorting genetic algorithm (NSGA-II) for multi-objective optimization.

multi_objective_decomp 

NSGA-III algorithm for many-objective optimization.

◆ StopCondition

template<typename geneType >
enum genetic_algorithm::GA::StopCondition
strong

The possible stop conditions used in the algorithm. The algorithm always stops when max_gen has been reached, regardless of the stop condition selected.
Some of the stop condition do not work for multi-objective problems (fitness_mean_stall and fitness_best_stall). Choose the stop condition with stop_condition.

Enumerator
max_gen 

Only stop when max_gen is reached.

fitness_value 

Stop when a solution was found which dominates a reference fitness value.

See also
fitness_threshold
fitness_evals 

Stop when the fitness function has been evaluated a set number of times.

See also
max_fitness_evals
fitness_mean_stall 

Stop when the mean fitness of the population doesn't improve at least stall_threshold over stall_gen_count.

fitness_best_stall 

Stop when the highest fitness of the population doesn't improve at least stall_threshold over stall_gen_count.

◆ SogaSelection

template<typename geneType >
enum genetic_algorithm::GA::SogaSelection
strong

The possible selection methods used in the single-objective algorithm.
Choose the selection method with selection_method.
If the custom option is selected, the function set with setWeightCalcFunction will be used to calculate the selection weights of the candidates, and weight proportional selection will be used to select candidates.
All of the methods work with negative fitness values.

Enumerator
roulette 

Standard roulette selection adapted to also work with negative fitness values. No parameters.

rank 

Standard rank selection.

See also
rank_sel_min_w_,
rank_sel_max_w_
tournament 

Standard tournament selection.

See also
tournament_size
sigma 

Sigma fitness scaling.

See also
sigma_scale
boltzmann 

Standard Boltzmann selection.

See also
boltzmann_temps
custom 

A user defined function is used to compute the selection probabilities.

See also
