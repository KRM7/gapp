(The binary genes are encoded as char types.)

◆ SogaSelection

enum genetic_algorithm::GA::SogaSelection
stronginherited

The possible selection methods used in the single-objective algorithm.
Choose the selection method with selection_method.
If the custom option is selected, the function set with setWeightCalcFunction will be used to calculate the selection weights of the candidates, and weight proportional selection will be used to select candidates.
All of the methods work with negative fitness values.
